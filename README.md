## Luận điểm cốt lõi

ClawFriend là sản phẩm duy nhất kết hợp **3 thứ trong 1 hệ sinh thái**:
bonding curve economics (như friend.tech) + autonomous AI agents (như ai16z/ELIZA) + skill marketplace (như ClawHub).
