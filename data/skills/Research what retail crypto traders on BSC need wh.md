## 1. Typical Research Process (Step-by-Step)

**Active degen trader finds token via Telegram/Dexscreener new pairs**:

1. **Contract Verification** (1min): Copy CA from Dexscreener → paste BscScan → check "Contract verified?" + creator tx history (bundler?).[^8]
2. **Liquidity Lock** (1min): BscScan → token holders → Team Finance/PinkSale links → verify Unicrypt/PinkSale locker (time/amount).[^1]
3. **Dev Wallet** (2min): BscScan "Holders" → sort top holders → check dev wallet sells/buys → DeBank profile for multi-token rugs.[^9]
4. **Holder Concentration** (2min): BubbleMaps.io → check top 10 holders >20%? → BscScan "Top Holders" for snipers/bots.[^1]
5. **Volume Authenticity** (2min): Dexscreener "Trades" tab → look for bot wash (100+ tx/min, equal buys/sells) → DEXTools "Wash Score".[^7]
6. **Social Health** (2min): Twitter/Telegram links from website → check member growth, shill quality → LunarCrush free social score.[^10]

**Total**: 10-15min/token → **20-50 tokens/week** for active degens.
