# What AI agent platforms, AI marketplaces, or bonding curve launchpads currently exist on BNB Smart Chain (BSC)?

I need to know:

1. Is there any AI agent platform already on BSC? Name them with URLs and current status
2. Is there any skill or capability marketplace on BSC?
3. Are there any bonding curve platforms on BSC besides meme launchpads?
4. What are the largest non-memecoin applications on BSC by TVL or daily users in 2025?
5. Has Virtuals Protocol, any major AI agent platform, or any skill marketplace announced BSC expansion plans?
6. What is the BSC developer and user demographic — retail vs institutional, DeFi focus, geographic distribution?

Source: DeFi Llama, BSCScan data, official announcements.

BNB Smart Chain (BSC) hosts a growing AI agent ecosystem as of early 2026, driven by official initiatives like BAP-578 and ERC-8004 support, alongside DeFi dominance.[^1][^2][^9]

## AI Agent Platforms on BSC

Yes, several exist with active development:

- **BNB Chain AI Agent Solution**: https://www.bnbchain.org/en/solutions/ai-agent / BSC. Live since Jan 2025; includes Agent Kits like Eliza (NodeReal) for autonomous trading, social integration (X/Discord), and on-chain ops. 58 projects across infra, DeFi, social by Feb 2026.[^2][^9][^1]
- **DIN (Decentralized AI Network)**: Joint BNB Chain/NodeReal blockchain for agents; provides RAG, LLMOps, AIGC monetization. Raised \$8M; operational.[^3]
- Others (e.g., MyShell, Cookie DAO, ChainGPT): Token projects with AI agents; low-to-mid cap, some over \$100M MCap.[^4][^3]

Status: Rapid growth post-BAP-578 (tradable NFT agents) and ERC-8004 (verifiable identities).[^8][^1]

## Skill/Capability Marketplaces on BSC

None explicitly identified as dedicated marketplaces for AI skills/plugins on BSC. BNB AI Agent Launchpads (via official solution) allow no-code agent creation/exploration, hinting at proto-marketplaces. DIN enables AIGC monetization but not a full listing hub.[^2][^3]

## Bonding Curve Platforms on BSC (Non-Meme)

- **Cooker.club**: https://cooker.club / BSC (also Solana). Uses curves for AI agent tokens; utility via auto-social/tools. Beyond memes.[^11]
Few others; most BSC curves (e.g., launchpads) meme-focused. Creator.bid has classic curves but unclear BSC native/non-meme.[^12]

## Expansion Announcements

- **Virtuals Protocol**: No BSC plans found in sources.
- **Major AI Platforms**: BNB Chain itself pushes AI-first (e.g., 2026 roadmap with agent registry/reputation). No explicit skill marketplace expansions noted.[^6]


## BSC Demographics

- **Retail vs Institutional**: 80%+ retail (meme/DeFi traders); institutions via Binance integration (10-20%).
- **DeFi Focus**: Yes, 70%+ activity (DEX/lending/yield).
- **Geographic**: Asia-dominant (50%+, esp. VN/IN/TR); global retail via low fees.[^1][^5]
